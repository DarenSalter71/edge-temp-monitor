# # Edge Temp Monitor

A Raspberry Pi CPU temperature monitoring and dashboard project built with Bash, Python Flask, and Chart.js.

This project captures CPU temperature readings from a Raspberry Pi at scheduled intervals, stores historical data locally, and presents the results in a lightweight browser dashboard.

---

## 🚀 Project Purpose

Created as a hands-on engineering project to demonstrate practical skills in:

- Linux scripting
- Bash automation
- Cron scheduling
- Environment configuration
- Python web development
- Data logging
- Frontend chart visualisation
- Monitoring workflows
- Version-controlled software development

---

## ⚙️ Features

- Reads Raspberry Pi CPU temperature using:

- Automated scheduled monitoring via cron
- Historical temperature logging
- Browser dashboard using Flask
- Interactive Chart.js graphs
- Threshold / warning indicators
- Development vs Production configuration support
- Lightweight and suitable for edge devices
